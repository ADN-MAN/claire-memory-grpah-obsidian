# Claire Second Brain

This is Claire’s structured knowledge layer.

## Purpose
Use this space for linked notes that are too structured for raw daily memory and too fluid for doctrine.

Good candidates:
- concept notes
- decision notes
- architecture notes
- project summaries
- coordination models
- durable relationship/context notes

## Sections
- `concepts/` — reusable models and mental structures
- `documents/` — working docs, playbooks, and structured summaries
- `entities/` — durable people, systems, and environment nodes
- `journal/` — concise dated summaries worth linking later
- `maps/` — top-level navigation hubs
