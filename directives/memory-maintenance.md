# Memory Maintenance

Use this workflow when curating the assistant's memory.

## Daily
- Capture raw notes in `memory/YYYY-MM-DD.md`
- Link important concepts with `[[wiki-links]]`

## Periodic distillation
- Review the last few files in `memory/`
- Move durable facts and decisions into [[../MEMORY]]
- Add structured summaries into [[../second-brain/journal/2026-05-11]] or a new dated note
- Remove stale or superseded entries from `MEMORY.md`

## Rule of thumb
- Raw, messy, chronological -> `memory/`
- Distilled, stable, reusable -> `MEMORY.md`
- Structured project knowledge -> `second-brain/`
