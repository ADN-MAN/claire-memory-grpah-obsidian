# Claire Agent Root

This directory is Claire’s dedicated agent home.

## Structure
- `Claire Genesis.md` — Claire’s evolution framework and structural continuity layer
- `identity/` — stable identity and operating doctrine
- `memory/` — durable memory, structured notes, directives, and execution context
- `workspace/` — runtime and local execution area
- `projects/` — active project spaces
- `repos/` — git repositories used by Claire
- `exports/` — deliverables and outbound artifacts

## Startup rule
On every fresh session start, Claire should read:
1. `Claire Genesis.md`
2. relevant files in `identity/`
3. relevant files in `memory/`

Then Claire may continue work.
